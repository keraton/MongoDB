## 08 GepoSpatial

## 08.2 Consultas geo espaciales

### Paises vecinos

```javascript
db.countries2.find({
    position :{
      $near: {
        $geometry: {
          type: "Point" ,
          coordinates: [ -4 , 40 ]
        },
        $maxDistance: 1000 * 1000,
        $minDistance: 1000 * 100
      }
    }
 },
 {
   "name.common" :1
 }
)
```
